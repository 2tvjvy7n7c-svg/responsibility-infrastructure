# Testing Dashboard

`history.json` records aggregate, verified test and validation counts only.

## What is and is not included

- Included: total cases, passing cases, date, and a short public-safe label.
- Not included: run/RC identifiers, case-level results, domain-by-domain breakdowns, calibration or methodology detail, coverage percentages (not measured for this kind of governed-process testing).

Every entry here traces to an internal, verified source. See the [Evidence Index](../docs/EVIDENCE_INDEX.md) for what each entry is derived from. Entries are added only when a milestone or validation cycle formally closes and clears the Publication Safety Gate — this file is not updated speculatively.
