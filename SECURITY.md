# Security

## Scope of this repository

This repository is a **public claim ledger** (research, high-level milestones, status, evidence maps). It does **not** host product production systems, customer data processing services, or private Doghouse runtime source.

## Reporting

If you believe this public repository contains:

- accidentally published secrets or credentials,
- personal data that should not be public,
- or other sensitive material that violates responsible disclosure,

contact: **doug@doghouseventures.com**

Please do **not** open a public GitHub issue that pastes secrets or exploit detail.

## What this is not

This file is not a vulnerability disclosure program for private Doghouse, Investment Vault, or Proof Ready implementation. Those systems are not published here. Speculation about private security design is out of scope for this repository (see [CONTRIBUTING.md](CONTRIBUTING.md)).
