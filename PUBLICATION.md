## Purpose of this repository

This repository is Dog House Ventures’ **public claim ledger**. It establishes a transparent public record of:

- the responsibility-infrastructure thesis and its authorship,
- its initial publication date and version history,
- its abstract and citation metadata,
- future revisions or related papers,
- **high-level engineering, product, validation, and governance milestones** admitted under the [Public Claim Ledger Charter](docs/PUBLIC_CLAIM_LEDGER_CHARTER.md),
- **program status** with explicit non-claims,
- and a repository-wide **claim→evidence** map with evidence classes.

The Git commit history preserves changes over time. The tagged GitHub release preserves Version 1.0 as a fixed **thesis** publication milestone.

## Scope of Version 1.0 (thesis)

Version 1.0 presents the working hypothesis that consequential AI-assisted work requires responsibility infrastructure connecting:

- authority,
- evidence,
- review,
- decisions,
- adaptive procedures,
- institutional memory,
- human approval,
- and explicit acceptance of responsibility.

Doghouse is mentioned as the experimental governed operating environment in which the process continues to be studied.

This publication does not disclose proprietary internal architecture, source code, prompts, schemas, security design, or implementation mechanisms.
